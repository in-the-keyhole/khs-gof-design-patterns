khs-gof-design-patterns
=======================

The Gang of Four desing patterns book, whas a seminal piece of work that helped introduce OO to the software development world in the 90's.  Understanding these patterns will help your own software designs. 

Checkout [this](https://github.com/in-the-keyhole/khs-gof-design-patterns/blob/master/Design%20Pattern%20Course.pdf) PDF for a walkthrough of the GOF patterns...Exercises are located in the `exercises` folder and exercise source solutions are in the `src` folder.
